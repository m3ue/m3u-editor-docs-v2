# Quick Actions

Quick Actions are pre-defined prompts that appear as clickable buttons inside the Copilot chat window. They let you trigger common workflows with a single click instead of typing the same request repeatedly.

---

## Setting Up Quick Actions

1. Go to **Preferences → AI Copilot**.
2. Scroll to the **Quick Actions** section.
3. Click **Add Quick Action**.
4. Fill in the fields:
   - **Label** — The button text shown in the chat window (e.g. `Find a channel`)
   - **Prompt** — The message sent to the AI when the button is clicked (e.g. `Help me find a channel by name`)
5. Click **Save**.

Your buttons will appear immediately the next time the Copilot is opened.

---

## Example Quick Actions

Here are some useful Quick Actions to get you started:

| Label | Prompt |
|---|---|
| Find a channel | Help me find a channel by name |
| Show active streams | List all currently active streams |
| Check proxy status | What is the current proxy status? |
| Add an EPG source | Help me add a new EPG source |
| Search the docs | Search the documentation for... |
| What can you do? | List all the tools and actions available to you |

Feel free to customise the prompts for your own workflow.

---

## Tips

- **Keep labels short** — they appear as compact buttons, so a few words work best (e.g. `Find channel`, not `Click here to search for a channel`).
- **Make prompts specific** — a more detailed prompt produces a faster, more accurate response (e.g. `Show me all playlists with proxy enabled` rather than just `Show playlists`).
- **Use them as shortcuts** — Quick Actions are ideal for tasks you repeat daily, like checking stream counts or looking up a channel.
- **Order matters** — buttons are displayed in the order they are added. Put your most-used actions first.
